# iQ™ Command Reference

_All iQ™ slash commands are grouped below by category. For privacy and terms, see [PRIVACY.md](./PRIVACY.md) and [TOS.md](./TOS.md)._

---

## Economy

- `/balance` — View your balance.
- `/daily` — Daily currency bonus.
- `/gamble` — Play to win or lose currency.
- `/slots` — Slot machine game.
- `/bankrobbery` — Risky group timed event.
- `/reflexes` — Quick reflex mini-game.
- `/work` — Simulated jobs with animated progress.
- `/beg` — Limited daily request for currency.
- `/transfer` — Send currency to another user.
- `/leaderboard` — Global economy leaderboard.
- `/profile` — Personal economy page.

---

## Leveling

- `/lv` — See your current level.
- `/rep` — Give rep to other users.

---

## Games & Fun

- `/8balliq` — Magic 8-ball (visual responses).
- `/coin` — Coin flip.
- `/quote` — Random quote.
- `/joke` — Random joke.
- `/fact` — Random fact.
- `/hug` — Send a hug.
- `/horoscope` — Daily sign-based reading.

---

## Utility

- `/ping` — Response time (with color feedback).
- `/avatar` — Show member’s avatar.
- `/userinfo` — Extended profile information.
- `/serverinfo` — Information about this server.
- `/membercount` — Server member stats.
- `/web` — Web search helper.
- `/weather` — Local weather look-up.
- `/define` — Dictionary feature.
- `/translate` — Multilingual support.
- `/calculate` — Simple math.
- `/timer` — Set a reminder.
- `/afk` — Set away-from-keyboard status.

---

## Moderation

- `/mute` — Mute member.
- `/kick` — Remove user from server.
- `/softban` — Ban/unban to clear posts.
- `/unban` — Unban user.
- `/warn` — Issue a warning.
- `/warnings` — View warnings.
- `/clearwarns` — Clear all warnings.
- `/lock` — Restrict channel.
- `/unlock` — Release channel restrictions.
- `/purge` — Bulk message removal.
- `/role` — Adjust user’s roles.
- `/report` — Escalate member to staff.
- `/setreportchannel` — Direct reports channel.
- `/raidclean` — Cleanup after mass join/attacks.
- `/raidstop` — End raid alert mode.

---

## Server Management

- `/announce` — Channel-wide broadcast.
- `/sticky` — Add sticky messages.
- `/unsticky` — Remove stickies.
- `/membercount` — Display current server count.

---

## RSS Feeds

- `/rssstatus` — Check all feeds’ status.
- `/rsslogs` — Show feed logs.
- `/rsscheck` — Verify feed operation.
- `/rssclear` — Remove all feeds.
- `/rsserrors` — List RSS problems.

---

## Bot Admin

- `/setstatus` — Change iQ status message.
- `/resync` — Manual command reload.
